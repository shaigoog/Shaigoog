# فكرتك تطبيق | Fikratak App

تطبيق أندرويد عربي/إنجليزي لاستقبال طلبات تصميم التطبيقات المخصصة بطريقة منظمة. يختار العميل نشاطه، يجيب عن أسئلة تتغير حسب المجال، يراجع ملخص الطلب، ثم يرسله مباشرة عبر واتساب.

## النسخة الحالية 1.0.0

- واجهة عربية RTL وإنجليزية LTR.
- اختيار 8 أنواع من الأنشطة.
- أسئلة ذكية متغيرة حسب النشاط.
- بيانات العميل والشركة والدولة والمدينة.
- كود مسوّق لمعرفة مصدر العميل.
- ملخص كامل قبل الإرسال.
- إنشاء رقم طلب تلقائي.
- إرسال واتساب أو مشاركة عبر أي تطبيق.
- إعداد GitHub Actions لاستخراج APK وAAB.

## بيانات التواصل

عدّل الملف:

`app/src/main/java/com/fikratak/app/AppConfig.kt`

رقم واتساب يجب أن يكون بمفتاح الدولة ومن دون `+` أو مسافات.
